Основные методы QPushButton

* setText(QString text): Устанавливает текст на кнопке.
* text(): Возвращает текущий текст на кнопке.
* setIcon(QIcon icon): Устанавливает иконку для кнопки.
* icon(): Возвращает текущую иконку кнопки.
* setIconSize(QSize size): Устанавливает размер иконки.
* iconSize(): Возвращает размер иконки.
* setFlat(bool flat): Устанавливает кнопку "плоской", то есть без видимой рамки.
* isFlat(): Возвращает true, если кнопка плоская, иначе false.
* setChecked(bool checked): Устанавливает состояние кнопки как "включенное" (checked) или "выключенное" (unchecked).
* isChecked(): Возвращает true, если кнопка "включена", иначе false.
* setDefault(bool default): Делает кнопку "по умолчанию", то есть она будет активирована при нажатии на Enter.
* isDefault(): Возвращает true, если кнопка "по умолчанию", иначе false.
* setAutoDefault(bool autoDefault): Включает или отключает автоматическое определение кнопки "по умолчанию" при нажатии на Enter.
* isAutoDefault(): Возвращает true, если автоматическое определение кнопки "по умолчанию" включено, иначе false.
* setMenu(QMenu* menu): Устанавливает контекстное меню для кнопки.
* menu(): Возвращает контекстное меню кнопки.
* setEnabled(bool enabled): Включает или отключает кнопку.
* isEnabled(): Возвращает true, если кнопка включена, иначе false.
* setVisible(bool visible): Скрывает или отображает кнопку.
* isVisible(): Возвращает true, если кнопка видима, иначе false.
* setGeometry(QRect rect): Устанавливает размер и положение кнопки.
* geometry(): Возвращает прямоугольник, представляющий размер и положение кнопки.
* move(int x, int y): Перемещает кнопку в заданное положение.
* resize(int width, int height): Изменяет размер кнопки.
* show(): Отображает кнопку.
* hide(): Скрывает кнопку.
* setToolTip(QString text): Устанавливает всплывающую подсказку для кнопки.
* toolTip(): Возвращает всплывающую подсказку кнопки.
* setStyleSheet(QString styleSheet): Устанавливает стиль CSS для кнопки.
* styleSheet(): Возвращает текущий стиль CSS кнопки.
* setShortcut(QKeySequence shortcut): Устанавливает горячую клавишу для кнопки.
* shortcut(): Возвращает текущую горячую клавишу для кнопки.
* setWhatsThis(QString text): Устанавливает всплывающую справку для кнопки.
* whatsThis(): Возвращает всплывающую справку для кнопки.

Сигналы QPushButton

* clicked(): Сигнал, который генерируется при нажатии на кнопку.
* pressed(): Сигнал, который генерируется при нажатии на кнопку мышью.
* released(): Сигнал, который генерируется при отпускании кнопки мышью.

* setMovie(QMovie* movie): Устанавливает анимированное изображение (QMovie) для QLabel.
* movie(): Возвращает анимированное изображение QLabel.
* setScaledContents(bool scaled): Включает или отключает автоматическую масштабирование содержимого QLabel для соответствия его размеру.
* setBuddy(QWidget* buddy): Привязывает QLabel к другому виджету (buddy), что позволяет использовать горячую клавишу для фокусировки на привязанном виджете.
* buddy(): Возвращает виджет, к которому привязан QLabel.
* setPalette(QPalette palette): Устанавливает палитру цветов для QLabel.
* palette(): Возвращает палитру цветов QLabel.
* setFixedWidth(int width): Устанавливает фиксированную ширину QLabel.
* setFixedHeight(int height): Устанавливает фиксированную высоту QLabel.
* setFont(QFont font): Устанавливает шрифт для текста в QLabel.
* font(): Возвращает текущий шрифт QLabel.
* setStyleSheet(QString styleSheet): Устанавливает стиль CSS для QLabel.
* styleSheet(): Возвращает текущий стиль CSS QLabel.
* clear(): Очищает содержимое QLabel.
* setEnabled(bool enabled): Включает или отключает QLabel.
* isEnabled(): Возвращает true, если QLabel включен, иначе false.
* setVisible(bool visible): Скрывает или отображает QLabel.
* isVisible(): Возвращает true, если QLabel виден, иначе false.
* setGeometry(QRect rect): Устанавливает размер и положение QLabel.
* geometry(): Возвращает прямоугольник, представляющий размер и положение QLabel.
* move(int x, int y): Перемещает QLabel в заданное положение.
* resize(int width, int height): Изменяет размер QLabel.
* show(): Отображает QLabel.
* hide(): Скрывает QLabel.
* setToolTip(QString text): Устанавливает всплывающую подсказку для QLabel.
* toolTip(): Возвращает всплывающую подсказку QLabel.
* setWhatsThis(QString text): Устанавливает всплывающую справку для QLabel.
* whatsThis(): Возвращает всплывающую справку для QLabel.

Сигналы QLabel:

* linkActivated(QString link): Сигнал, который генерируется при клике на ссылку в тексте QLabel.

Основные методы QLineEdit

setText(QString text): Устанавливает текст в поле ввода.
text(): Возвращает текущий текст в поле ввода.
setPlaceholderText(QString placeholder): Устанавливает текст-заполнитель (placeholder), который отображается в поле ввода, пока оно пустое.
placeholderText(): Возвращает текст-заполнитель.
setEchoMode(QLineEdit::EchoMode mode): Устанавливает режим отображения текста в поле ввода. Например, QLineEdit::Normal для обычного отображения, QLineEdit::Password для отображения звездочек вместо текста.
echoMode(): Возвращает текущий режим отображения текста.
setReadOnly(bool readOnly): Включает или отключает возможность редактирования поля ввода.
isReadOnly(): Возвращает true, если поле ввода доступно только для чтения, иначе false.
setAlignment(Qt::Alignment alignment): Устанавливает выравнивание текста в поле ввода. Например, Qt::AlignCenter для центрального выравнивания, Qt::AlignLeft для левого.
setMaxLength(int maxLength): Устанавливает максимальное количество символов, которое может быть введено в поле ввода.
maxLength(): Возвращает максимальное количество символов.
setValidator(QValidator* validator): Устанавливает валидатор, который ограничивает вводимые символы.
validator(): Возвращает текущий валидатор.
setClearButtonEnabled(bool enabled): Включает или отключает кнопку очистки (clear button), которая отображается рядом с полем ввода.
isClearButtonEnabled(): Возвращает true, если кнопка очистки включена, иначе false.
selectAll(): Выбирает весь текст в поле ввода.
setCursorPosition(int position): Устанавливает позицию курсора в поле ввода.
cursorPosition(): Возвращает текущую позицию курсора.
setSelection(int start, int end): Выбирает часть текста в поле ввода.
selectionStart(): Возвращает начальную позицию выделения текста.
selectionEnd(): Возвращает конечную позицию выделения текста.
setModified(bool modified): Устанавливает состояние "модифицировано" (modified) для поля ввода.
isModified(): Возвращает true, если поле ввода было модифицировано, иначе false.
setPalette(QPalette palette): Устанавливает палитру цветов для QLineEdit.
palette(): Возвращает палитру цветов QLineEdit.
setFont(QFont font): Устанавливает шрифт для текста в QLineEdit.
font(): Возвращает текущий шрифт QLineEdit.
setStyleSheet(QString styleSheet): Устанавливает стиль CSS для QLineEdit.
styleSheet(): Возвращает текущий стиль CSS QLineEdit.
setEnabled(bool enabled): Включает или отключает QLineEdit.
isEnabled(): Возвращает true, если QLineEdit включен, иначе false.
setVisible(bool visible): Скрывает или отображает QLineEdit.

setGeometry(QRect rect): Устанавливает размер и положение QLineEdit.
geometry(): Возвращает прямоугольник, представляющий размер и положение QLineEdit.
move(int x, int y): Перемещает QLineEdit в заданное положение.
resize(int width, int height): Изменяет размер QLineEdit.
show(): Отображает QLineEdit.

setToolTip(QString text): Устанавливает всплывающую подсказку для QLineEdit.
